❓ Ajuda — FreeNameConvention

1
📁 Adicione uma pasta
Clique em + Adicionar Pasta na página de Pastas (ou Ctrl+N). Escolha o diretório que deseja monitorar. O assistente abre automaticamente.
2
📐 Escolha uma normativa (ou crie seu padrão)
O assistente exibe as normativas disponíveis. Escolha a do seu setor — ex.: NF-e (SEFAZ) para documentos fiscais, ISO 9001 para qualidade. Se nenhuma servir, escolha Padrão próprio e use o Construtor de Padrão.
3
▶ Inicie o Guardião
Clique em ▶ Iniciar na barra superior (ou Ctrl+G). O Guardião passa a monitorar todas as pastas habilitadas em tempo real usando chokidar.
4
⚠️ Quando um arquivo não conformes é detectado
Uma janela nativa do Windows aparece com o nome do arquivo, exemplos válidos e uma sugestão automática de nome em conformidade. Você pode renomear com 1 clique, mover para quarentena ou ignorar. Alertas também chegam por notificação do Windows e/ou e-mail.
💡 Dica rápida: Use a página Validar para testar o nome de um arquivo antes de salvá-lo na pasta monitorada. O sistema sugere automaticamente o nome correto.

Separadores: Use - hífen, _ underscore ou . ponto entre os tokens. O separador faz parte do padrão — DANFE-{CNPJ} só aceita hífen entre DANFE e o CNPJ.

O Construtor de Padrão permite criar regras de nomenclatura visualmente, sem digitar tokens manualmente.

1
Abra o Construtor
Na pasta, clique em Editar → Opções avançadas → Construtor de Padrão, ou via menu Ferramentas → Construtor de Padrão Personalizado.
2
Adicione elementos
Clique em + Adicionar elemento. Escolha o tipo (CNPJ, Data, Número, Texto, Lista de opções, etc.). Cada bloco pode ter um separador - _ . configurável.
3
Configure cada elemento
  • Número sequencial: clique no botão 4d para alterar a quantidade de dígitos
  • Lista de opções: clique em ⋯ para definir os valores possíveis, ex.: NF|NFS|NFC
  • Hash / Texto tamanho fixo: clique em ⋯ para definir N
  • Texto fixo (literal): digite diretamente no campo do bloco
4
Modo avançado (opcional)
Ative Modo avançado para editar o padrão técnico diretamente. Útil para copiar/colar padrões prontos de documentos da normativa.

O Guardião bloqueia a edição de arquivos já existentes?

Não. O Guardião monitora apenas arquivos novos (evento add do sistema). Arquivos já existentes na pasta no momento em que o Guardião inicia não são verificados.

O que é a pasta _NONCOMPLIANT?

Quando a ação é Mover para quarentena ou Bloquear, arquivos fora do padrão são movidos automaticamente para esta subpasta. A pasta é criada com acesso restrito ao grupo Administrators e SYSTEM usando icacls. Você pode definir uma pasta de quarentena personalizada em cada pasta monitorada.

Posso ter múltiplos padrões por pasta?

Sim! Uma pasta pode ter vêrios padrões ativos. Um arquivo é válido se corresponder a qualquer um dos padrões. Útil quando a pasta recebe arquivos de tipos diferentes, como PDF e XML.

Como filtrar apenas alguns tipos de arquivo?

No modal Editar da pasta, na seção Extensões de arquivo monitoradas, adicione as extensões desejadas (ex.: .pdf, .xml). Arquivos de outros tipos serão ignorados pelo Guardião.

Como exportar e importar as configurações?

Use o menu Arquivo → Exportar Configuração para salvar todas as pastas e padrões em um arquivo .json. Use Arquivo → Importar Configuração para carregar em outro computador.

O CNPJ deve ter pontuação no nome do arquivo?

Não. O token {CNPJ} espera 14 dígitos sem pontos, barras ou hífens: 12345678000199. Nomes de arquivo não devem conter / pois esse caractere é reservado pelo sistema.

Como receber alerta por e-mail?

Em Configurações → Alertas e Notificações, ative Alertas por e-mail e preencha o servidor SMTP. Para Gmail, ative a verificação em 2 etapas e gere uma Senha de App em minhaconta.google.com/seguranca.

Posso usar o Guardião sem o aplicativo aberto?

Sim. O aplicativo fica na bandeja do sistema (system tray) mesmo com a janela fechada. Configure Iniciar com o Windows em Configurações para que o Guardião inicie automaticamente.
